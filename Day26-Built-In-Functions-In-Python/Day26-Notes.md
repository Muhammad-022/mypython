## Common Built-in Functions
### 1. len()
Returns the number of items in a list or characters in a string.

### 2. type()
Tells the data type of a variable.

### 3. max()
Returns the largest value.

### 4. min()
Returns the smallest value.

### 5. sum()
Returns the total of numbers in a list.

### Example Use
Built-in functions are mostly used with lists, numbers, and strings.
