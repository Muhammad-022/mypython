# Day 2: print() Function in Python

## Introduction
The `print()` function is one of the most basic and important functions in Python.  
It is used to display output on the screen.

Almost every Python program uses the `print()` function to show messages or results.

## What is print() Function?
The `print()` function displays text or values to the console.

## Uses of print() Function

The `print()` function is used to:

- Display messages  
- Show results  
- Debug programs  
- Communicate with the user  

Example:
```python
print("Hello, Python")
